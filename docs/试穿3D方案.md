# 3D 试穿方案（GLB + 挂点 + 正反抠图）

## 架构

```
用户抠图（正面/背面） → 衣橱 { srcFront, srcBack }
                              ↓
试穿页 tryon-xr → GLB 人模 + 平面 mesh 贴图（按槽位挂点）
                              ↓
失败 / 低版本 → model-tryon 2D Canvas 降级
```

## 已实现

| 模块 | 说明 |
|------|------|
| `components/tryon-xr/` | xr-frame 加载 GLB + 槽位平面贴图 |
| `utils/tryon3dAnchors.js` | 男女模挂点坐标（可后续改为 GLB 空节点） |
| `utils/wardrobeItem.js` | `srcFront` / `srcBack` 归一化 |
| `utils/cloudFileUrl.js` | cloud:// → https |
| `config/cdn.js` | `getModel3dUrl(gender)` |
| `pages/tryon/` | 默认 3D，右上角可切 2D |
| `add-clothing` | 上衣/下装/套装/内搭支持录入背面 |
