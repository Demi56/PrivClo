# 3D 数字人模型接入指南

## 一、方案对比

| 方案 | 难度 | 特点 | 基础库要求 |
|------|------|------|------------|
| **xr-frame（推荐）** | ⭐⭐ | 微信官方方案，XML 声明式，支持拖拽/旋转 | 2.27.1+，推荐 2.32.0+ |
| **Three.js** | ⭐⭐⭐ | 灵活可控，需较多代码 | 支持 WebGL 即可 |

---

## 二、方案一：xr-frame（微信官方）

### 1. 准备 3D 模型文件

- **格式**：`.glb` 或 `.gltf`（推荐 .glb，单文件方便）
- **建议规格**：面数适中、骨骼/蒙皮符合限制（见官方文档）
- **获取途径**：
  - [GLB下载网](https://glbxz.com/) - 人物模型
  - [CGMOL 摩尔网](https://www.cgmol.com/) - 按 gltf/glb 筛选
  - [爱给网](https://www.aigei.com/3d/model/) - 3D 模型库
  - [Sketchfab](https://sketchfab.com/) - 筛选 downloadable + gltf

### 2. 放置模型文件

将 `.glb` 文件放入项目，例如：
```
小程序2.0/
├── models/
│   ├── avatar-female.glb   ← 女生数字人
│   └── avatar-male.glb    ← 男生数字人
```

### 3. 在 app.json 中启用 xr-frame

```json
{
  "plugins": {
    "xr-frame": {
      "version": "0.0.0",
      "provider": "wx-xr-frame"
    }
  }
}
```

### 4. 基础库版本

在微信开发者工具中，将「详情」→「本地设置」→「调试基础库」设为 **2.32.0** 或更高。

### 5. 使用示例（在页面中）

```xml
<xr-scene id="xr-scene">
  <xr-asset-load asset-id="avatar" src="/models/avatar-female.glb" type="gltf" />
  <xr-node node-id="model-root" position="0 0 0" scale="1 1 1">
    <xr-gltf model="avatar" />
  </xr-node>
  <xr-camera position="0 1.2 2.5" target="0 1 0" />
</xr-scene>
```

## 三、方案二：Three.js（threejs-miniprogram）

### 1. 安装依赖

```bash
cd 小程序2.0
npm install threejs-miniprogram
```

### 2. 构建 npm

在微信开发者工具中：**工具** → **构建 npm**。

### 3. 在页面中使用 canvas

```xml
<canvas type="webgl" id="webgl-canvas" style="width:100%;height:500px" />
```

### 4. 在 JS 中加载模型

```javascript
import { createScopedThreejs } from 'threejs-miniprogram'

// 在 onReady 中
const canvas = await ... // createSelectorQuery 获取
const THREE = createScopedThreejs(canvas)
const loader = new THREE.GLTFLoader()
loader.load('/models/avatar.glb', (gltf) => {
  scene.add(gltf.scene)
  // 添加 OrbitControls 实现拖拽旋转
})
```

## 四、推荐实施步骤（xr-frame）

1. 下载或制作一个 glb 格式的 3D 数字人模型  
2. 放入 `小程序2.0/models/` 目录  
3. 在 app.json 中添加 xr-frame 插件配置  
4. 将模特页面中间区域改为 xr-scene 渲染（或使用下方提供的组件）  
5. 真机调试验证（xr-frame 在部分模拟器上可能不完整）

---

## 五、在项目中启用 3D 模型

### 1. 已提供的文件

- `components/model3d/`：xr-frame 3D 展示组件
- `models/`：用于存放 `.glb` 模型文件

### 2. 启用步骤（4 步）

1. **放入模型**：将 `.glb` 文件放入 `小程序2.0/models/`
   - 女生：`avatar-female.glb`
   - 男生：`avatar-male.glb`

2. **【重要】预处理模型**：使用 [xr-frame-toolkit](https://developers.weixin.qq.com/miniprogram/dev/component/xr-frame/tools/toolkit.html#gltf%E4%BC%98%E5%8C%96) 对 glb 进行优化，生成适配 xr-frame 的版本。未预处理的模型可能出现 `Texture of undefined`、`animator of undefined` 等错误。

3. **开启 3D**：在 `pages/model/model.js` 的 `data` 中，把 `use3D: false` 改为 `use3D: true`

4. **编译运行**：重新编译，调试基础库选择 **2.32.0** 及以上，建议真机预览

---

## 六、注意事项

- **包体大小**：单 glb 建议控制在 5MB 内，过大影响加载
- **真机测试**：xr-frame 需在真机上充分测试
- **兼容性**：xr-frame 不支持 PC 小程序，仅 iOS/Android

### 常见错误排查

若出现 `TypeError: Cannot read property 'Texture' of undefined` 或 `animator of undefined`：
1. 使用 **xr-frame-toolkit** 对 glb 做预处理（GLTF 优化），导出后再用
2. 或更换更简单的模型（纹理少、无复杂动画）
3. 临时可用 `use3D: false` 保持 2D 占位展示
