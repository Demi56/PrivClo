# 试穿渲染系统使用说明

## 架构概览

- **bodyAnchors.js** - 身体锚点与分区配置
- **clothingWarper.js** - 衣物变形（仿射变换）
- **layerRenderer.js** - 多图层、边缘羽化
- **model-tryon** 组件 - 整合上述模块完成试穿绘制

## 1. 身体锚点 (bodyAnchors.js)

### 配置说明

坐标基于参考画布 **375×720**，与 `clothingPositions` 一致。

```javascript
const { getAttachPoint, getClothingAdjustment } = require('./utils/bodyAnchors.js')

// 获取附着点（像素）
const pt = getAttachPoint('top', 'female', canvasW, canvasH)

// 衣物微调（在 clothingAdjustments 中配置）
const adj = getClothingAdjustment(url, 'top')
// { x, y, scale, skewX, skewY }
```

### 自定义微调

在 `bodyAnchors.js` 的 `clothingAdjustments` 中新增：

```javascript
clothingAdjustments['top_001'] = { x: -2, y: 0, scale: 1.02 }
clothingAdjustments['bottom_002'] = { x: 0, y: 3, scale: 0.98 }
```

key 为图片文件名（不含扩展名），`default` 为兜底。

## 2. 衣物变形 (clothingWarper.js)

- `computeClothingDrawParams()` - 按附着点、槽位、微调计算绘制参数
- `drawWithTransform()` - 支持 skew 的绘制
- `getAffineTransform()` / `warpToQuad()` - 仿射/四边形变形（进阶）

## 3. 多图层 (layerRenderer.js)

### 单 URL（当前默认）

```javascript
outfit: { top: 'https://xxx/top.png', bottom: '...' }
```

### 多图层格式

```javascript
outfit: {
  top: {
    base: 'https://xxx/top_base.png',
    shadow: 'https://xxx/top_shadow.png',
    highlight: 'https://xxx/top_highlight.png'
  }
}
```

绘制顺序：base → shadow → highlight。

## 4. model-tryon 组件

传入 `gender`、`outfit` 即可，组件会：

1. 加载模特底图（contain 保持比例）
2. 按 z-order 绘制衣物
3. 使用附着点 + 微调定位
4. 边缘羽化 (shadowBlur ≈ 2)

## 5. 调优建议

- **附着点**：在 `bodyAnchors.js` 中微调 `attachPoints` 的 x、y
- **衣物偏移**：在 `clothingAdjustments` 中按图片配置 x、y、scale
- **多图层**：为需要立体感的部位准备 shadow、highlight 图，使用对象格式传入
